목차
1. 프로젝트 컨셉
2. 벤치마킹 사이트
3. 주요기능






주요기능 
1. 회원가입
  - 자신의 정보와 이상형을 입력한다.
2. 매칭
  - 자신이 원하는 조건에 맞는 이성을 매칭해준다.
  - 조건에 맞는 회원 리스트를 확인 할 수 있다.
  - 자신에게 호감을 표현한 회원을 확인 할 수 있다.
  - 매일 신규 이성회원들을 추천 해준다.
  - 자신이 호감을 표현한 상대가 자신에게도 호감을 표현하면 1:1 채팅을 걸어준다.
3. 커뮤니티
  - 익명 실시간 채팅
  - 매칭 후기, 연애초보들을 위한 연애 상담
  - 번개모임
